# Gestor de Projetos

App web (HTML/JS puro, sem build step) para gestão de custos, ROI e tarefas
de projetos, com login e permissões por usuário via Supabase.

- Banco de dados: Supabase (Postgres + Auth + RLS)
- Hospedagem: GitHub Pages

## Status

- [x] Fase 1: login, permissões por usuário, cadastro de projetos
- [ ] Fase 2: Financeiro (custos/receita + ROI consolidado)
- [ ] Fase 3: Tarefas (Kanban)
- [ ] Fase 4: Configurações completas (categorias de custo, colunas do kanban)
